## ✅ Supported / Recommended for Sonos

- UPnP (Universal Plug and Play) — Enabled
- Wireless mode: b / g / n
- 2.4 GHz Wi‑Fi enabled
- Channel width: 20 MHz
- IGMP Snooping — ON
- Separate SSIDs for 2.4 GHz and 5 GHz
- Standard home network (non‑guest)
- Simple WPA2 security
- Direct connection to main router
- Ethernet connection (when available)

---

## ❌ Not Supported / Causes Issues with Sonos


### 🧠 Key Rule to Remember
If a Wi‑Fi feature claims to be **smart, fast, secure, or optimized**,  
it’s probably **bad for Sonos**.
Keep the network **simple, predictable, and boring**.
