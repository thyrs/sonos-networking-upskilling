
# Sorting Activity Network Connection Types


<!-- alt+z  FOR word wrap -->


Instruction:
Sort each example into the correct network model: Client–Server or Peer‑to‑Peer.

Examples:
|                                                          |                                                                       |
| -------------------------------------------------------- | --------------------------------------------------------------------- |
| Printing from a laptop to a USB‑connected printer        | Peer‑to‑Peer (P2P) — direct connection, no server.                    |
| Sending photos using AirDrop                             | Peer‑to‑Peer (P2P) — devices connect directly.                        |
| Spotify streaming music on a phone                       | Client–Server — music comes from Spotify servers.                     |
| Accessing Google Drive from a browser                    | Client–Server — files live on Google servers.                         |
| Two PCs playing a LAN game without internet              | Peer‑to‑Peer (P2P) — machines connect directly.                       |
| Email app checking messages from Gmail                   | Client–Server — email stored on mail servers.                         |
| Sharing files with a USB flash drive                     | Peer‑to‑Peer (P2P) — data moves device to device.                     |
| Watching YouTube on a smart TV                           | Client–Server — video streamed from YouTube servers.                  |
| Two phones chatting using offline Wi‑Fi Direct           | Peer‑to‑Peer (P2P) — no router, no server.                            |
| Logging into an online banking website                   | Client–Server — bank systems run on secure servers.                   |
| Multiplayer game hosted by one player’s PC               | Peer‑to‑Peer (P2P) — one player acts as the host, no official server. |
| Using Zoom for a video call                              | Client–Server — Zoom’s servers manage the call.                       |
| Smart TV streaming Netflix                               | Client–Server — content delivered from Netflix servers.               |
| Sharing files via Bluetooth between a laptop and a phone | Peer‑to‑Peer (P2P) — short‑range, direct transfer.                    |
| Weather app showing today’s forecast                     | Client–Server — app pulls data from weather servers.                  |
| Bluetooth printer and phone                              | *Peer‑to‑Peer*                                                        |
| File sharing between two laptops                         | *Peer‑to‑Peer*                                                        |
| Two smartphones sending files via Bluetooth              | *Peer‑to‑Peer*                                                        |
| Hulu streaming to an Apple TV                            | *Client–Server*                                                       |
| Web browser accessing a website                          | *Client–Server*                                                       |
